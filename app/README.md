# Property Portal Backend

FastAPI backend for Property Listing Portal.

## Features

- JWT Authentication
- Role Based Authorization
- Property CRUD APIs
- Pagination
- Search & Filter
- Dashboard Stats

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- JWT
- Pydantic

## Run Project

Install dependencies:

pip install -r requirements.txt

Run server:

uvicorn app.main:app --reload
